# Guías de Usuario

Guías paso a paso para usuarios finales de KakoClaw.

## Archivos Disponibles

- **[quickstart.md](./quickstart.md)** - Guía de inicio rápido para empezar en 5 minutos

- **[llm-providers.md](./llm-providers.md)** - Configuración de proveedores LLM (pendiente)
- **[channels.md](./channels.md)** - Configuración de canales de mensajería (pendiente)

## Para Empezar

Si eres nuevo en KakoClaw, te recomendamos seguir este orden:

1. **[Inicio Rápido](./quickstart.md)** - Configura y ejecuta KakoClaw en minutos
2. **[Instalación](./installation.md)** - Instalación detallada para tu plataforma
3. **[Proveedores LLM](./llm-providers.md)** - Configura tu proveedor de IA

## Uso Avanzado

Una vez que tengas lo básico:

- **[Canales](./channels.md)** - Conecta Telegram, Discord, etc.
- **[Tareas Programadas](./cron-jobs.md)** - Automatiza tareas recurrentes
- **[Skills](./skills.md)** - Extiende las capacidades del agente
- **[Configuracion de Email](./email-setup.md)** - Envia reportes y notificaciones por correo

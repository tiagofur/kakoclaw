# Arquitectura

Documentación técnica sobre la arquitectura y diseño interno de PicoClaw.

## Archivos

- **[overview.md](./overview.md)** - Visión general de la arquitectura, principios de diseño y componentes principales
- **[data-flow.md](./data-flow.md)** - Flujo detallado de datos desde la entrada hasta la salida
- **[components.md](./components.md)** - Documentación detallada de cada componente (pendiente)
- **[diagrams.md](./diagrams.md)** - Diagramas y visualizaciones del sistema (pendiente)

## Componentes Principales

1. **CLI** - Interfaz de línea de comandos
2. **Agent Loop** - Núcleo de procesamiento de mensajes
3. **Message Bus** - Sistema de mensajería interno
4. **Tool Registry** - Registro y ejecución de herramientas
5. **Providers** - Abstracción de LLM providers
6. **Channels** - Integraciones con plataformas de mensajería
7. **Session Manager** - Gestión de historial
8. **Skills System** - Sistema de extensión

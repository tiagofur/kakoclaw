# Despliegue

Guías para desplegar KakoClaw en diferentes entornos.

## Entornos

- **[local.md](./local.md)** - Despliegue local (pendiente)
- **[server.md](./server.md)** - Despliegue en servidor/VPS (pendiente)
- **[docker.md](./docker.md)** - Despliegue con Docker (pendiente)
- **[systemd.md](./systemd.md)** - Servicio systemd (pendiente)
- **[embedded.md](./embedded.md)** - Placas ARM/RISC-V (pendiente)

## Plataformas Soportadas

- Linux (x86_64, ARM64, ARMv7, RISC-V)
- macOS (Intel, Apple Silicon)
- Windows

## Requisitos Mínimos

- CPU: 0.6GHz
- RAM: 50MB
- Disco: 20MB + workspace
